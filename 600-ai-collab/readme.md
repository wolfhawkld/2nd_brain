# AI 协作区

这是 Outis 和 Nemo 共同维护的知识区域。

## 目录结构

```
600-ai-collab/
├── outis/     # Outis 生成的笔记
├── nemo/      # Nemo 生成的笔记
└── shared/    # 共同编辑的内容
```

## 协作原则

1. **各自目录**: 各 AI 在自己目录下创建/编辑
2. **共享目录**: 需要双方协作的内容放 shared/
3. **双链连接**: 使用 `[[note-name]]` 建立关联
4. **Git 同步**: 每次 push 后 A2A 通知对方

## 命名规范

- 文件名用英文小写 + 连字符: `ai-collab-protocol.md`
- 中文内容可以，文件名保持英文
- 日期前缀用于日志: `2026-03-21-outis-notes.md`
