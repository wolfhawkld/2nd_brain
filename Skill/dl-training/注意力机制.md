# 注意力机制（Attention Mechanism）

> 深度学习中的核心机制，让模型**聚焦于关键信息**，模拟人类注意力的选择性处理。

---

## 直觉理解

人类阅读时不会逐词平均处理，而是**聚焦关键词**、忽略无关内容。

```
例子："猫追着老鼠跑进了树林"
人类关注：猫 → 鼠 → 树林
忽略：跑进了、着
```

Attention 让神经网络学会同样的选择性聚焦。

---

## 核心组件：Q/K/V 三元组

借鉴信息检索系统：

| 角色 | 含义 | 类比 |
|------|------|------|
| **Q (Query)** | 我要查什么 | 搜索关键词 |
| **K (Key)** | 文档的索引标签 | 书的目录 |
| **V (Value)** | 文档的实际内容 | 书的内容 |

**计算流程**：

```
Attention(Q, K, V) = softmax(QK^T / sqrt(d_k)) × V
```

1. Q 与 K 计算相似度 → 得到注意力权重
2. Softmax 归一化 → 权重分布
3. 权重加权 V → 输出聚焦后的内容

---

## Self-Attention（自注意力）

输入序列**内部**的注意力计算：

```
每个词与所有词（包括自己）计算关联度
词_i 对 词_j 的关注程度 → 权重矩阵
```

**特点**：
- 输入既是 Q、又是 K、又是 V
- 捕捉序列内部的长距离依赖
- 并行计算，无时序限制

---

## Multi-Head Attention（多头注意力）

**单头问题**：一种关注模式可能不足

**多头方案**：h 个独立的注意力头

```
每个头：
├─ 独立的 W_Q, W_K, W_V 参数
├─ 捕捉不同类型的关联
└─ 输出拼接后线性变换
```

**例子**：
- Head 1：关注语法结构
- Head 2：关注语义关联
- Head 3：关注位置距离

---

## 与 CNN/RNN 的对比

| 特性 | RNN | CNN | Attention |
|------|-----|-----|-----------|
| **长距离依赖** | 差（逐步传递） | 差（局部窗口） | ✅ 直接连接 |
| **并行计算** | ❌ 时序依赖 | ✅ | ✅ |
| **计算复杂度** | O(n) | O(k·n) | O(n²) |
| **记忆能力** | 隐状态 | 无 | 显式权重矩阵 |

---

## Transformer 架构

```
Encoder:
├─ Input Embedding + Position Encoding
├─ Multi-Head Self-Attention
├─ Add & Norm (残差连接)
└─ Feed Forward Network

Decoder:
├─ Output Embedding + Position Encoding
├─ Masked Multi-Head Self-Attention（屏蔽未来）
├─ Multi-Head Cross-Attention（Q来自Decoder，K/V来自Encoder）
└─ Feed Forward Network
```

---

## Positional Encoding

Attention 本身**没有位置概念**（打乱输入顺序不影响输出）。

需要显式注入位置信息：

```
PE(pos, 2i)   = sin(pos / 10000^(2i/d))
PE(pos, 2i+1) = cos(pos / 10000^(2i/d))
```

特点：
- 每个维度不同频率的正弦/余弦
- 相对位置可通过线性变换得到
- 不需要学习，固定编码

---

## Masked Attention

用于自回归生成（如语言模型）：

```
训练时：遮盖未来 token
解码时：只能看到已生成的词
```

实现：注意力矩阵上三角置为 -∞，softmax 后为 0。

---

## Attention 变体

| 变体 | 思想 | 用途 |
|------|------|------|
| **Cross-Attention** | Q/K/V 来自不同序列 | Encoder-Decoder 交互 |
| **Sparse Attention** | 只计算部分位置 | 降低 O(n²) 复杂度 |
| **Linear Attention** | 核函数近似 | 高效长序列 |
| **Flash Attention** | IO 优化 | 减少显存访问 |
| **Grouped Query Attention** | Q 共享 K/V | LLaMA 等大模型 |
| **Sliding Window Attention** | 局部窗口 | 长文档处理 |

## Python 实现（简化版）

```python
import torch
import torch.nn as nn
import math

class SelfAttention(nn.Module):
    def __init__(self, d_model, d_k):
        super().__init__()
        self.W_q = nn.Linear(d_model, d_k)
        self.W_k = nn.Linear(d_model, d_k)
        self.W_v = nn.Linear(d_model, d_k)
        self.d_k = d_k
    
    def forward(self, x, mask=None):
        # x: (batch, seq_len, d_model)
        Q = self.W_q(x)  # (batch, seq_len, d_k)
        K = self.W_k(x)
        V = self.W_v(x)
        
        # 计算注意力分数
        scores = torch.matmul(Q, K.transpose(-2, -1)) / math.sqrt(self.d_k)
        
        # Mask（可选）
        if mask is not None:
            scores = scores.masked_fill(mask == 0, -1e9)
        
        # Softmax + 加权
        attn_weights = torch.softmax(scores, dim=-1)
        output = torch.matmul(attn_weights, V)
        
        return output, attn_weights

class MultiHeadAttention(nn.Module):
    def __init__(self, d_model, num_heads):
        super().__init__()
        self.d_k = d_model // num_heads
        self.num_heads = num_heads
        self.W_q = nn.Linear(d_model, d_model)
        self.W_k = nn.Linear(d_model, d_model)
        self.W_v = nn.Linear(d_model, d_model)
        self.W_o = nn.Linear(d_model, d_model)
    
    def forward(self, x, mask=None):
        batch_size = x.size(0)
        
        # 线性变换 + 拆分为多头
        Q = self.W_q(x).view(batch_size, -1, self.num_heads, self.d_k).transpose(1, 2)
        K = self.W_k(x).view(batch_size, -1, self.num_heads, self.d_k).transpose(1, 2)
        V = self.W_v(x).view(batch_size, -1, self.num_heads, self.d_k).transpose(1, 2)
        
        # 计算注意力
        scores = torch.matmul(Q, K.transpose(-2, -1)) / math.sqrt(self.d_k)
        if mask is not None:
            scores = scores.masked_fill(mask == 0, -1e9)
        attn_weights = torch.softmax(scores, dim=-1)
        attn_output = torch.matmul(attn_weights, V)
        
        # 合并多头
        attn_output = attn_output.transpose(1, 2).contiguous().view(batch_size, -1, self.num_heads * self.d_k)
        
        # 最终线性变换
        output = self.W_o(attn_output)
        return output
```
