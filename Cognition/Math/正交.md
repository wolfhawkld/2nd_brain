# 正交（Orthogonality）

> 线性代数中的核心概念，两向量**互相垂直**或矩阵**保持向量长度**的性质。在深度学习中用于初始化、优化和表示学习。

---

## 直觉理解

物理世界中的"垂直"：

```
地板 ⟂ 墙壁 — 90度夹角
x轴 ⟂ y轴 — 直角坐标系
```

正交 = 90度关系 = **方向独立、互不影响**

---

## 向量正交

### 定义

两个向量 u、v 正交，当：

```
u · v = 0  （内积为零）
```

几何意义：夹角为90°

### 性质

| 性质 | 说明 |
|------|------|
| **独立性** | 正交向量互不包含对方的分量 |
| **最远距离** | 同长度向量中，正交时距离最大 |
| **投影为零** | u在v上的投影长度为0 |

### 正交向量组

一组向量 {v₁, v₂, ..., vₙ} 正交：

```
v_i · v_j = 0  (i ≠ j)
```

若还满足 ||v_i|| = 1，则称为**正交归一化**（Orthonormal）。

---

## 正交矩阵

### 定义

矩阵 Q 是正交矩阵，当：

```
Q^T Q = I  或  Q Q^T = I
```

等价条件：
- Q 的列向量相互正交且归一化
- Q 的行向量相互正交且归一化
- Q^T = Q^{-1}

### 性质

| 性质 | 公式 |
|------|------|
| **保长度** | ||Qx|| = ||x|| |
| **保角度** | (Qx) · (Qy) = x · y |
| **行列式** | det(Q) = ±1 |
| **逆矩阵** | Q^{-1} = Q^T（计算极快） |

### 应用

- **旋转矩阵**：2D/3D旋转
- **坐标变换**：FFT、小波变换
- **QR分解**：Q是正交矩阵

---

## 正交基

### 定义

向量空间的一组基 {e₁, e₂, ..., eₙ} 正交归一化：

```
e_i · e_j = δ_ij  （δ_ij是克罗内克符号）
```

### 优势

| 优势 | 说明 |
|------|------|
| **表示简洁** | 向量 x = Σ(x·e_i)e_i |
| **投影简单** | 在某个基向量上的投影 = 内积 |
| **独立计算** | 各分量互不影响 |

### 常见正交基

- 标准基：{(1,0,0), (0,1,0), (0,0,1)}
- Fourier基：{sin(kx), cos(kx)}
- 小波基：Haar、Daubechies等

## 深度学习中的应用

### 正交初始化

初始化权重矩阵为正交矩阵：

```python
import torch
W = torch.nn.init.orthogonal_(torch.empty(m, n))
```

**优势**：
- 防止梯度消失/爆炸（保长度）
- 加速初期收敛
- 信息流更稳定

### 正交正则化

损失函数中加入正交约束：

```
L_orth = ||W^T W - I||_F
```

**目的**：
- 防止特征间冗余
- 提升表示多样性
- 改善泛化能力

### Muon优化器

Muon 使用 Newton-Schulz 迭代对动量矩阵进行**正交化**：

```
msign(M) = M(M^T M)^{-1/2}
```

提取正交因子 U V^T（SVD极分解）。

详见 [[Muon优化器]]。

## 正交化方法

### Gram-Schmidt 正交化

将一组向量转化为正交向量组：

```
输入：{v₁, v₂, ..., vₙ}
输出：{u₁, u₂, ..., uₙ}（正交）

步骤：
u₁ = v₁
u₂ = v₂ - (v₂·u₁)/(u₁·u₁) × u₁
u₃ = v₃ - (v₃·u₁)/(u₁·u₁) × u₁ - (v₃·u₂)/(u₂·u₂) × u₂
...
```

### QR分解

任何矩阵 A 可分解为：

```
A = QR
Q：正交矩阵
R：上三角矩阵
```

### Newton-Schulz迭代

高效近似正交化（Muon优化器使用）：

```
迭代 X_{k+1} = X_k (3 - X_k^T X_k) / 2
收敛到正交矩阵
```
