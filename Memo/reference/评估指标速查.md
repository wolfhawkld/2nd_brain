# 评估指标速查

## 分类指标

| 指标 | 公式 | 说明 |
|-----|------|------|
| Accuracy | (TP+TN)/Total | 正确率 |
| Precision | TP/(TP+FP) | 精确率 |
| Recall | TP/(TP+FN) | 召回率 |
| F1 | 2PR/(P+R) | 调和平均 |
| AUC-ROC | 曲线下面积 | 排序能力 |

## 回归指标

| 指标 | 说明 |
|-----|------|
| MSE | 均方误差 |
| MAE | 平均绝对误差 |
| R² | 解释方差比例 |

## 生成指标

| 指标 | 应用 |
|-----|------|
| BLEU | 机器翻译 |
| ROUGE | 文本摘要 |
| Perplexity | 语言模型 |
| FID | 图像生成 |
