# 激活函数速查表

## 常用激活函数

| 名称 | 范围 | 应用场景 |
|-----|------|---------|
| ReLU | [0,+inf) | 隐藏层标配 |
| Sigmoid | (0,1) | 二分类输出 |
| Tanh | (-1,1) | RNN隐藏层 |
| GELU | (-inf,+inf) | Transformer |
| Softmax | (0,1),和=1 | 多分类输出 |

## 选择指南

- 隐藏层：ReLU / GELU
- 二分类输出：Sigmoid
- 多分类输出：Softmax
- RNN：Tanh
